# @milkdown/plugin-indent

Indent support for [milkdown](https://milkdown.dev/).

```typescript
import { Editor } from "@milkdown/kit/core";
import { indent } from "@milkdown/kit/plugin/indent";

Editor.make().use(indent).create();
```
